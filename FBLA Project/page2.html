Linking Basics

Use the <a> tag to create links. Here are some examples:

  • Link to a page in the same folder: <a href="page2.html">Go to Page 2</a>
  • Link to an external site: <a href="https://example.com" target="_blank" rel="noopener noreferrer">External Site</a>
  • Link to a section on the same page: <a href="#section-id">Jump to Section</a>

For more information, refer to the documentation.
